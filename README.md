# WeChatJumpGame

#### 项目介绍
微信"跳一跳"小游戏辅助

#### 依赖环境
包括但不限于：
1. anaconda 5 with python 3
2. tensorflow
3. keras
4. 安卓调试工具：adb

#### 使用说明

0. 训练数据下载：https://pan.baidu.com/s/1c2rrlra
1. clone项目
2. usb连接手机并打开调试模式，启动跳一跳，点击 开始游戏
3. 在文件夹下执行： python WeChatJumpGame.py
4. 待卷积网络训练完成即可自动开始。

本项目最高可达到7W分， 平均在3W分左右。
